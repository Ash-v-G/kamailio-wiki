====== Perl API for Kamailio v4.0.x ======

Module **perl** allows execution of Perl scripts from SIP router configuration file, providing access to SIP message which is processed at that time. Exported functions by **perl** are available in the Perl script as package Kamailio.

References:

  * **perl** module: [[http://kamailio.org/docs/modules/4.0.x/modules/app_perl.html|README]]
  * **Perl**: http://www.perl.org

Perl API documentation is part of module README.
