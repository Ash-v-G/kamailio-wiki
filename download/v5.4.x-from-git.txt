====== Download Sources of Kamailio v5.4.x from GIT Repository ======

Instructions to download sources from GIT repository, specific for stable branch 5.4 (release series v5.4.x).

===== Kamailio v5.4.x =====

Direct download of sources from GIT for latest Kamailio v5.4.x.

<code>
  git clone --depth --no-single-branch 1 https://github.com/kamailio/kamailio kamailio
  cd kamailio
  git checkout -b 5.4 origin/5.4
</code>
